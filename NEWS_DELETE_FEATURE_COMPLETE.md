# 任务结果删除能力说明

## 功能定位
该能力原本用于“新闻管理删除”，当前已归入 `AI 助手` 的 `任务结果` 管理能力。首版仍然主要删除新闻类结果，但命名和页面入口已经切换到平台化表达。

## 已实现内容
- 后端支持删除保存结果：`DELETE /api/news/saved/:id`
- 前端结果列表支持删除按钮和确认操作
- 删除后同步更新前端状态
- 删除失败时向用户返回错误提示

## 当前架构
### 后端
- Repository / Service / Controller / Route 四层链路已打通

### 前端
- 结果列表页支持删除操作
- 删除确认与加载态已接入

## 当前使用流程
1. 进入 `任务结果` 页面。
2. 找到要删除的内容卡片。
3. 点击删除按钮并确认。
4. 删除成功后内容从列表移除。

## 注意事项
- 当前结果实体仍复用 `SavedNews`。
- 后续若统一升级为通用 `TaskResult`，删除接口也应一起做平台化命名迁移。
- 本文档保留是为了说明删除链路已经完成，不再代表产品仍是“新闻管理系统”。
